# Correct Functional Bugs in Legacy JavaScript Code

This repository contains a legacy JavaScript utility module with multiple functional bugs. The `repository_before` directory contains the buggy code, while `repository_after` is reserved for the corrected implementation.

## Requirements

- [Docker](https://www.docker.com/get-started)
