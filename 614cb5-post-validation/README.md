# Post Validation with zod


## Commands

1. Run tests on repository_before codebase:
  ```sh
  
  docker compose run --rm test-after
  ```
2. Run tests on repository_after codebase:
  ```sh
  
  docker compose run --rm test-after
  ```
3. Run evaluation and generate reports:
  ```sh
  docker compose run --rm evaluate
  ```
