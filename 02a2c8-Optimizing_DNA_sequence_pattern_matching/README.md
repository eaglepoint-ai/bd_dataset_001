# DNA Sequence Pattern Matching

This repository demonstrates a DNA sequence pattern matching implementation using the Bitap (Shift-Or) algorithm. The `repository_before` directory represents the initial state, while `repository_after` contains the completed implementation.

## Requirements

- [Docker](https://www.docker.com/get-started)
- [Docker Compose](https://docs.docker.com/compose/)

## Quick Start

### 1. Run Tests for `repository_before`

```bash
docker compose run --rm test-before
```

### 2. Run Tests for `repository_after`

```bash
docker compose run --rm test-after
```

### 3. Run Evaluations

```bash
docker compose run --rm evaluation
```

## Projects

| Directory            | Description              |
| -------------------- | ------------------------ |
| `repository_before/` | Initial state            |
| `repository_after/`  | Completed implementation |
| `tests/`             | Test suite               |
| `evaluation/`        | Evaluation scripts       |
